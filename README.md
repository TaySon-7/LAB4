# MAI Casino

## Введение
В данном проекте реализовано казино с американской рулеткой, разными игроками, гусями. Проект позволяет игрокам покупать и продавать фишки, делать ставки  
взаимодействовать с балансом казино.

## Архитектура проекта

### Общая архитектура
Реализована функция симуляции, где, используя конкртеный seed, можно получать
одни и те же результаты.
Релизованы две пользовательские коллекции:
- ChipBalance: это баланс игрока в фишках, чтобы он появился, нужно сначала купить фишки.
- CasinoBalance: это баланс самого казино, который показывает сколько фишек каких цветов остается в наличии.
В проекте есть классы:
- Player: состоит из имени (name), баланса в фишках (chip_balance), баланса в долларах (money)
- Chip: игровая фишка, имеет цвет (color), цену за единицу (price), количесвтор в стопке (stack)
реализована возможность складывать фишки одного цвета через "+".
- Casino: основной класс, в котором происходят основные события: продажа, покупка фишек, ставки.

## Пользование

В файле main.py можно сыграть в казино от своего лица, задава баланс и имя
В файле simulation.py можно запустить 20 случайных событий и посмотреть, как происходит игра.
Весь функционал поддерживает seed.
